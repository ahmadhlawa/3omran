# حلول عمرانية ذكية | Smart Urban Solutions

نموذج أولي واجهة فرونت إند لمشروع معماري وبصري يقدمه مهندس معماري فلسطيني وصانع محتوى لرصد المشاكل الهندسية والعمرانية في القرى والبلدات الفلسطينية وتقديم مقترحات واقعية باستخدام الذكاء الاصطناعي.

---

## 🌟 الميزات الرئيسية (Key Features)

* **تصميم كامل باللغة العربية مع دعم الاتجاه RTL**: استخدام خط جوجل **Tajawal** وتنسيق كامل ملائم للهوية الفلسطينية والمعمارية.
* **بطاقات ثنائية الأبعاد 3D Flip Cards**: استعراض الوضع الحالي (قبل) والمقترح المعماري (بعد) بسلاسة مع دعم الأجهزة الكفية واللمس والوصولية عبر لوحة المفاتيح (`keyboard accessible`).
* **تصفية المشاكل حسب التصنيف**: إمكانية استعراض المشاكل حسب الأرصفة، الإنارة العامة، واجهات الأحياء، تنظيم الشوارع، مداخل البلدات، والفراغات العامة.
* **دعوة للمشاركة المجتمعية (Community Call to Action)**: قسم مخصص لتشجيع المواطنين على التقاط صور للمشاكل في أحيائهم وإرسالها عبر واتساب لتكون موضوع الحلول القادمة.
* **نظام معالجة صور مرن (Fallback Engine)**: في حال عدم توفر بعض الصور، تعطي المنصة عرضاً بصرياً أنيقاً ومحتوى هندسياً توضيحياً دون ظهور أي أيقونات مكسورة.

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **React 19**
* **TypeScript**
* **Vite**
* **Tailwind CSS v4**
* **Lucide React** (لخيار الأيقونات المعمارية)

## 🚀 تشغيل المشروع (How to Run)

لتشغيل المشروع محلياً:

```bash
# 1. تثبيت الحزم والمكتبات
npm install

# 2. تشغيل سيرفر التطوير
npm run dev
```

افتح المتصفح على الرابط: `http://localhost:3000`

## 🔗 تخصيص الروابط والصور (Replacing Assets & Links)

1. **الصور المعمارية**:
   * صورة البطل قبل: `/images/hero-before.webp`
   * صورة البطل بعد: `/images/hero-after.webp`
   * صور بطاقات المشاكل والحلول: داخل `/images/solutions/`
2. **رابط الواتساب**:
   * قم بتعديل متغير `whatsappUrl` في الملف `src/components/CommunitySection.tsx` لاستبدال `PHONE_NUMBER` برقمك الفعلي.
3. **فيديوهات الشرح**:
   * قم بتعديل قيمة `videoUrl` داخل `src/data/solutions.ts` لكل بطاقة بحسب رابط الفيديو على يوتيوب أو إنستغرام أو تيك توك.
4. **روابط التواصل الاجتماعي**:
   * قم بتعديل اسم المستخدم في `src/components/SocialLinks.tsx`.
